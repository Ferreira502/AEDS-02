# Questão 1 — Análise, recursão e fila circular — 6 pts

## 1(a) Análise dos blocos

Pelo que você escreveu, tinha um loop externo de 10 vezes e um loop interno que variava de 2 em 2, ou algo constante. Como o número de repetições não depende de `n`, a complexidade é constante.

Gabarito:


Se tivesse um loop que roda até `n`, aí seria `O(n)`. Mas “10 vezes” é constante.

## 1(b) Somatório triplo

Você escreveu algo tipo:

```text
Σ Σ Σ k² + 1
```

Se os três laços vão até `n`, e o termo interno depende de `k`, então:

```text
Σ(i=1 até n) Σ(j=1 até n) Σ(k=1 até n) (k² + 1)
```

O somatório interno é:

```text
Σ(k=1 até n) (k² + 1) = O(n³)
```

Porque:

```text
Σ k² = O(n³)
```

Mas ainda tem os laços de `i` e `j`, cada um rodando `n` vezes:

```text
O(n) * O(n) * O(n³) = O(n⁵)
```

Gabarito:

```text
O(n⁵)
```

Seu erro: colocou `O(k²)`. A resposta tem que sair em função de `n`, não de `k`.

## 1(c) Loop logarítmico

Se o código divide ou multiplica a variável por uma constante, tipo:

```java
while (n > 1) {
    n = n / 2;
}
```

Gabarito:

```text
O(log n)
```

Essa você acertou.

## 1(d) Indução / soma de potências de 3

A soma correta é:

```text
S_n = 3⁰ + 3¹ + 3² + ... + 3ⁿ
```

Fórmula fechada:

```text
S_n = (3^(n+1) - 1) / 2
```

### Base

Para `n = 0`:

```text
S_0 = 3⁰ = 1
```

Pela fórmula:

```text
(3^(0+1) - 1) / 2
= (3 - 1) / 2
= 1
```

Base verdadeira.

### Passo indutivo

Hipótese:

```text
S_n = (3^(n+1) - 1) / 2
```

Queremos provar:

```text
S_(n+1) = (3^(n+2) - 1) / 2
```

Começando:

```text
S_(n+1) = S_n + 3^(n+1)
```

Substituindo a hipótese:

```text
S_(n+1) = (3^(n+1) - 1) / 2 + 3^(n+1)
```

Colocando no mesmo denominador:

```text
S_(n+1) = (3^(n+1) - 1 + 2 * 3^(n+1)) / 2
```

```text
S_(n+1) = (3 * 3^(n+1) - 1) / 2
```

```text
S_(n+1) = (3^(n+2) - 1) / 2
```

Provado.

Seu erro: você chegou perto, mas usou fórmula sem o `-1` e marcou a base como falsa.

## 1(f) Insertion sort em fila circular

Seu código está certo para vetor normal, mas se a questão era fila circular, precisava considerar a posição real no vetor usando módulo.

Gabarito estilo fila circular:

```java
void insertionSort() {
    for (int i = 1; i < tamanho; i++) {
        int posI = (primeiro + i) % array.length;
        int tmp = array[posI];

        int j = i - 1;

        while (j >= 0) {
            int posJ = (primeiro + j) % array.length;
            int posJMais1 = (primeiro + j + 1) % array.length;

            if (array[posJ] > tmp) {
                array[posJMais1] = array[posJ];
                j--;
            } else {
                break;
            }
        }

        int posFinal = (primeiro + j + 1) % array.length;
        array[posFinal] = tmp;
    }
}
```

Seu erro: fez insertion sort de vetor simples. Conceito bom, mas faltou circularidade.

```java
private boolean estaNaMatriz(int x, CelulaMat inicio, int L) {
    CelulaMat linha = inicio;

    for (int i = 0; i < L; i++) {
        CelulaMat coluna = linha;

        for (int j = 0; j < L; j++) {
            if (coluna.elemento == x) {
                return true;

            coluna = coluna.dir;

        linha = linha.inf;

    return false;
```

## 2(d) Remover última coluna da matriz flexível

Uma forma correta:

```java
void removerUltimaColuna(CelulaMat inicio) {
    if (inicio == null) {
        return;
    }

    CelulaMat linha = inicio;

    while (linha != null) {
        CelulaMat coluna = linha;

        while (coluna.dir != null && coluna.dir.dir != null) {
            coluna = coluna.dir;
        }

        if (coluna.dir != null) {
            coluna.dir = null;
        }

        linha = linha.inf;
    }
}
```

Ideia: para cada linha, andar até a penúltima célula e fazer:

```java
penultima.dir = null;
```

Seu código foi na direção certa, mas misturou `linha`, `coluna`, `inf`, `dir` de um jeito que não remove corretamente a última coluna.

typedef struct CelulaDupla {
    int elemento;
    struct CelulaDupla* prox;
    struct CelulaDupla* ant;
} CelulaDupla;

CelulaDupla* primeiro;

void swap(CelulaDupla* a, CelulaDupla* b) {
    int tmp = a->elemento;
    a->elemento = b->elemento;
    b->elemento = tmp;
}

CelulaDupla* particionar(CelulaDupla* esq, CelulaDupla* dir) {
    int pivo = dir->elemento;
    CelulaDupla* i = esq->ant;

    for (CelulaDupla* j = esq; j != dir; j = j->prox) {
        if (j->elemento <= pivo) {
            if (i == esq->ant)
                i = esq;
            else
                i = i->prox;

            swap(i, j);
        }
    }

    if (i == esq->ant)
        i = esq;
    else
        i = i->prox;

    swap(i, dir);
    return i;
}

void quicksortRec(CelulaDupla* esq, CelulaDupla* dir) {
    if (esq != dir && esq != dir->prox) {
        CelulaDupla* p = particionar(esq, dir);
        quicksortRec(esq, p->ant);
        quicksortRec(p->prox, dir);
    }
}

void quicksort() {
    if (primeiro != NULL && primeiro->prox != primeiro && primeiro->prox != primeiro->ant) {
        quicksortRec(primeiro->prox, primeiro->ant);
    }
